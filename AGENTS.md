# Oscar Lugo — Portfolio

## Proyecto
Portfolio profesional de Oscar Lugo (Highway43). Editor de video, creador de contenido y diseñador visual.

## Deploy
- Push a `main` → GitHub Actions despliega automáticamente a GitHub Pages
- URL: https://oscarlugocar-lang.github.io/portfolio/

## Stack
- HTML + CSS + vanilla JS
- GitHub Pages + GitHub Actions
- GSAP + ScrollTrigger + Lenis smooth scroll

## Design System (MONOLOG × IL CAPO)
- --bg: #000000
- --text: #F5F5F0
- --accent: #D60001 (rojo)
- --border: rgba(255,255,255,.06)
- Font headings: Inter (uppercase, heavy)
- Font mono: DM Mono (labels mono)
- Custom cursor with bubble
- Smooth scroll with Lenis
